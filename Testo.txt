GL.SYSTEM PRINCIPLE
Determinism precedes amplification.
Structure precedes stochasticity.
Control precedes scale.
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


MISSION

Design a deterministic supervisory layer positioned
upstream of LLM invocation.

Objective:
Stabilize amplification.
Reduce regime chatter.
Bound stochastic escalation.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
FORMAL CORE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

INPUT:
    f(t)  → feature vector
    w     → weights, Σ|w_i| = 1

ENERGY:
    E_t = Σ w_i f_i(t)
    E_t ∈ [0,10]

STABILITY (EMA):
    S_t = αE_t + (1 − α)S_(t−1)

REGIME ENGINE:
    DROP      if S_t ∈ [0,3]
    STABLE    if S_t ∈ (3,7]
    SURGE     if S_t ∈ (7,10]

HYSTERESIS:
    Escalation requires N_up consecutive samples.
    De-escalation requires N_down consecutive samples.
    Cooldown prevents flip-flop transitions.

GATE:
    G(state) → {PASS, HOLD, DROP}
    Deterministic.
    Non-bypassable.
